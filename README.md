# SaaP Financial Intelligence Engine

## You need TWO terminal windows open at the same time.

---

## Terminal 1 — Backend

```
cd ~/Downloads/saap/backend
chmod +x start_backend.sh
./start_backend.sh
```

Leave this running. You'll see: "Backend running at http://localhost:8000"

---

## Terminal 2 — Frontend

```
cd ~/Downloads/saap/frontend
chmod +x start_frontend.sh
./start_frontend.sh
```

Leave this running. You'll see: "Frontend running at http://localhost:3000"

---

## Open the app

Go to: http://localhost:3000

You'll see the full dashboard with charts, KPI cards, and AI insights.
Switch between Romania / USA / Germany using the dropdown.
